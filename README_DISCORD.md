# Discord Бот для Dota 2 GSI

Discord бот для получения информации о текущем матче Dota 2 с ссылками на Dotabuff.

## Установка

1. Установите зависимости:
```bash
uv sync
```

2. Создайте Discord бота:
   - Перейдите на https://discord.com/developers/applications
   - Создайте новое приложение
   - Перейдите в раздел "Bot"
   - Создайте бота и скопируйте токен
   - Включите "Message Content Intent" в настройках бота

3. Пригласите бота на сервер:
   - В разделе "OAuth2" -> "URL Generator"
   - Выберите scope: `bot`
   - Выберите permissions: `Send Messages`, `Read Message History`
   - Скопируйте URL и откройте в браузере для приглашения

4. Настройте токен:
   - Создайте файл `.env` в корне проекта
   - Добавьте строку: `DISCORD_TOKEN=ваш_токен_бота`
   - Или установите переменную окружения:
     ```bash
     # Windows (PowerShell)
     $env:DISCORD_TOKEN="ваш_токен_бота"
     
     # Linux/macOS
     export DISCORD_TOKEN="ваш_токен_бота"
     ```

## Запуск

```bash
uv run python discord_bot.py
```

## Команды

### `!match`
Выводит список всех игроков текущего матча в формате:
```
Ник - https://www.dotabuff.com/players/76561198218419015
```

### `!ping`
Проверка работы бота (ответит "Pong!")

## Требования

- Запущенный GSI сервер (`uv run python src/server.py`)
- Активный матч в Dota 2
- Файлы матчей в папке `output/`

## Примечания

- Бот использует последний сохраненный файл матча
- Если матч не найден, бот сообщит об ошибке
- Сообщения автоматически разбиваются на части, если превышают лимит Discord (2000 символов)
